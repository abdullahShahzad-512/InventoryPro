# Inventory Management System (Semester Project)

Simple ASP.NET Core MVC application for inventory management, part of a semester project.

## Requirements
- .NET SDK 8.0 or newer

## Quick start
1. Restore packages:

   ```powershell
   dotnet restore
   ```

2. Build:

   ```powershell
   dotnet build
   ```

3. Run:

   ```powershell
   dotnet run
   ```

The app will typically run at `https://localhost:5001` when launched.
